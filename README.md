#### Простая игра в гонки на Pixi.js

## Stack
Typescript + Pixi.js + Redux Toolkit + Vite + i18n-js + Howler


## Запуск

### Установить npm-зависимости
```
npm i
```
### Запустить сервер
```
npm run dev
```
### Открыть игру в браузере
http://localhost:5173/
