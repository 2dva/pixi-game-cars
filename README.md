# Small Cars
Простая игра в гонки на Pixi.js

Управление: w,a,s,d,space
## Демо
https://pixi-game-cars-deploy.vercel.app/


## Запуск

### Установить npm-зависимости
```
npm i
```
### Запустить сервер
```
npm run dev
```
### Открыть игру в браузере
http://localhost:5173/

# Stack

Pixi.js + Typescript + Vite
